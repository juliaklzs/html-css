Parágrafos e Quebras de Linha

O parágrafo fica entre o par de tags <p> e </p>. e pode ser escrito de qualquer forma

Independente de paragrafos no código, o texto aparecerá inteiro no site, porque html é uma linguagem baseada em marcações.
Para quebrar o texto, basta usar a tag <br>.

Para criar espaçamentos entre os textos, basta utilizar o css.
